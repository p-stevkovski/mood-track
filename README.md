# MoodTrack

A privacy-first web app to track your daily wellbeing. Answer 4 quick questions about mood, stress, sleep, and activity to get personalized insights and visualize your wellness trends.

![Status](https://img.shields.io/badge/status-active-blue)
![License](https://img.shields.io/badge/license-MIT-green)
![Version](https://img.shields.io/badge/version-1.0.0-blue)

---

## ✨ Features

- **Daily Survey** - 4 quick questions (~30 seconds), personalized feedback and tips
- **Data Visualization** - 7-day trend chart with mood and calmness patterns
- **Export Data** - Download CSV for backup or analysis
- **100% Private** - All data stored locally, no servers or tracking
- **Responsive** - Works on mobile, tablet, and desktop
- **No Dependencies** - Pure HTML, CSS, JavaScript

---

## 🚀 Quick Start

1. Clone the repository:
   ```bash
   git clone https://github.com/p-stevkovski/mood-track.git
   cd mood-track
   ```

2. Open `index.html` in your browser - that's it!

3. Deploy online:
   - **GitHub Pages:** Enable in repo settings
   - **Netlify/Vercel:** Connect your repo for automatic deployment
   - **Any web host:** Upload `index.html`

---

## 📋 How It Works

1. **Take Survey** - Answer questions about mood, stress, sleep, and activity
2. **Get Feedback** - Receive personalized wellness message and tips
3. **View History** - See your 7-day trend in the chart below
4. **Export or Delete** - Download CSV backup or clear your data

---

## 🔒 Data Storage

- Data stored in browser's localStorage (100% on your device)
- No servers, no cloud, no tracking
- Maximum 30 entries stored (auto-cleanup)
- Clear browser cache to delete data
